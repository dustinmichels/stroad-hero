## Format Code

### Setup

```sh
python -m venv venv
source venv/bin/activate
pip install gdtoolkit
```

### Format

```sh
./format
```

### Notes

When pulling code, there may be troubles with the global cache.

You can try deleting the contents of `./godot` (but not the folder) and reloading the project.
